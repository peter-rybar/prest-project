# Bufet

Bufet project.

## Development environment setup

`npm install`

## Build project

`npm run build`

## Test project

`npm run test`

## Dist project

`npm run dist`

## Clean project

`npm run clean`

## Clean all project

`npm run clean:all`

## Run development server

`npm run server:watch`

## Run production server

`npm run server:cluster`
