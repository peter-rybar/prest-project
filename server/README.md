# pREST Project

pREST project.

## Development environment setup

`npm install`

## Build project

`npm run build`

## Test project

`npm run test`

## Clean project

`npm run clean`

## Clean all project

`npm run clean:all`

## Run development server

`npm run server:watch`

## Run production server

`npm run server:cluster`

## Run production server pm2

`npm run server:pm2`
